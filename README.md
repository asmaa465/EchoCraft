# 🎙️ EchoCraft — AI-Powered Personalized Content Generator

> Transform your CV and references into polished, audience-ready LinkedIn posts — automatically.

---

## 📌 Overview

**EchoCraft** is an end-to-end AI automation pipeline built with **n8n** that generates personalized professional content for platforms like LinkedIn, Facebook, and Medium.

The user fills out a form with their background, purpose, and preferences. EchoCraft takes care of everything else — reading their CV, processing references, generating a tailored post using AI, creating a matching image, and delivering the final result directly to their inbox.

---

## ✨ Features

- 📄 **CV Parsing** — Uploads and converts CV documents to text via AWS Lambda
- 🎥 **YouTube Reference Support** — Extracts transcripts from YouTube links as reference material
- 📁 **Document Reference Support** — Accepts uploaded project/document files as references
- 🤖 **AI Content Generation** — Uses Google Gemini 2.5 Pro with ROC-T-TOC prompting framework
- 🎨 **AI Image Generation** — Generates a matching LinkedIn-ready image via HuggingFace FLUX.1
- 💾 **User Profile Storage** — Saves client profiles to AWS DynamoDB
- 📧 **Automated Email Delivery** — Sends the generated post + image directly via Gmail API
- 🌍 **Multilingual Output** — Supports English and Egyptian Arabic output

## 🔄 How It Works

1. **User fills a form** with their name, academic degree, work scope, purpose, audience, tone, writing style, and preferred output language
2. **CV is uploaded** → converted to plain text via AWS Lambda
3. **Reference is provided** → either a YouTube URL (transcript extracted) or an uploaded document (converted to text)
4. **All data is merged** into a structured prompt using the **ROC-T-TOC framework** (Role, Objective, Context, Task, Tone, Output, Constraints)
5. **Google Gemini 2.5 Pro** generates a personalized post (100–500 words depending on format)
6. **Image description** is generated by a second AI agent, then passed to **FLUX.1** to generate a visual
7. **User profile** is stored in AWS DynamoDB
8. **Final post + image** is sent to the user's email via Gmail API

---

## 🧠 Prompting — ROC-T-TOC Framework

EchoCraft uses a structured prompting approach for consistent, high-quality output:

| Letter | Meaning |
|---|---|
| **R** | Role — Define the AI's persona |
| **O** | Objective — What the content should achieve |
| **C** | Context — User's background, CV, and references |
| **T** | Task — Specific writing instructions |
| **T** | Tone — Formal, casual, inspiring, etc. |
| **O** | Output — Format, language, word count |
| **C** | Constraints — No jargon, no sensitive data, platform-ready |

## 🚀 Getting Started

### Prerequisites
- [n8n](https://n8n.io/) instance (self-hosted or cloud)
- Google Gemini API key
- HuggingFace API token
- AWS account (Lambda + DynamoDB)
- Gmail OAuth2 credentials
- RapidAPI account (YouTube Transcript API)

### Import the Workflow
1. Open your n8n instance
2. Go to **Workflows → Import**
3. Upload `EchoCraf_Project.json`
4. Configure credentials for each service
5. Activate the workflow
